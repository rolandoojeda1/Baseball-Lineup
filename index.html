4 Pitch Count & Rest Tracker

Rest days owed after today (SPLL Regulation VI(d)): 1–20 pitches → 0 days · 21–35 → 1 day · 36–50 → 2 days · 51–65 → 3 days · 66+ → 4 days. A pitcher can never throw on 3 consecutive calendar days. This tool only tracks the current game — note these rest requirements yourself for upcoming games, since pitch history isn't saved between sessions.
